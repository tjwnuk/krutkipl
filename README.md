## Install

1. Install golang

```https://go.dev/doc/install```

2. Clone this repo

```git clone git@github.com:tjwnuk/krutkipl.git```

3. Run the server

```go run main.go```

The server will start. Optionally you can build binary

```go build -o krutkipl```

and run it
```./krutkipl```
